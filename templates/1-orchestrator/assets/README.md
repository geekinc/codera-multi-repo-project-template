# assets/

Files uploaded via the Codera orchestrator card land in `uploads/` and are
committed here so Claude Code in the dev container can reference screenshots,
designs, and examples. Run `git pull` to see newly uploaded files.

## Limits

- **Per-file cap:** 6 MB (CodeCommit) / 25 MB (GitHub).

## Layout

```
assets/
  uploads/   <- files uploaded through the orchestrator card (committed)
```
